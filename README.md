# Panda_Pohua_bot

#№ Мы спроектировали Telegram-бота, который становится единой точкой доступа к разным видам контента. Его главная особенность — модульность и интеграция со сторонними сервисами:

- **Рассказы и шутки: Бот обращается к внешним API, чтобы всегда иметь свежий и разнообразный контент.

- **Китайская кухня: Рецепты также поступают из специализированного внешнего API, что гарантирует их точность и разнообразие.
